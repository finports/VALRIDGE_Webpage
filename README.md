# [VALRIDGE with FinPorts](https://valridge.finports.com)

주식회사 [핀포츠](https://finports.com)의 VALRIDGE는 주식시장의 동향을 분석하여 시스템 트레이딩 알고리즘과 운용전략 등을 제공하며 알고리즘 트레이딩 서비스를 제공할 예정입니다.
